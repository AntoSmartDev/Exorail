# READY_CHECKLIST.md

## Setup

- [ ] readiness status and gate agree
- [ ] required blueprint entries are sufficient
- [ ] required blueprint rows map input evidence to reachable canonical sources
- [ ] Product, Architecture, Engineering and Roadmap have sufficient canonical coverage
- [ ] no required fact or accepted decision survives only in chat or an intake snapshot
- [ ] an aggregate blueprint does not compete with its split canonical sources
- [ ] blocking decisions are resolved
- [ ] acceptance and verification are concrete

## Task closure

- [ ] worktree changes are classified
- [ ] acceptance is evaluated
- [ ] verification evidence is recorded
- [ ] result and cursor are updated
- [ ] subsequent work is reviewed
- [ ] workflow validator passes when installed

## Delivery Unit closure

- [ ] aggregate review is complete
- [ ] follow-ups are explicit
- [ ] documentation and readiness are aligned
- [ ] integration decision is recorded
- [ ] archive decision is recorded

## Rule

Keep only checks that are mechanical and repeatedly useful. Semantic judgment belongs in readiness and reviews.
